Configuration/Setup of DA Project. 
**********************************

This is an utility library that has various functionalities for date, time and other methods.

Overview of using the DA library
================================
1. Create a Python virtual environment and  install the required packages.
2. git clone and run python setup.py install
3. pip install <DA package name>


Functionality Overview
======================
1. Datetime and related functionalities.
2. CSV features.
3. SQL functionalities for different databases.
4. Run the different testsuites

    python zzdates.py

5. Refer to code for more details
